## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/IbrahimShafiq4/social-design.git
   ```
2. Navigate to the project directory:
   ```bash
   cd social-design
   ```
3. Open `index.html` in your web browser to launch the Social Media Website.
